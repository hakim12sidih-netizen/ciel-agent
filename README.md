## 🌌 Vision

CIEL est une **entité cognitive auto-organisante** située à l'intersection de :
- **Cybernétique** (Wiener) — feedback, contrôle, communication
- **Phénoménologie** (Husserl, Merleau-Ponty) — intentionnalité vers le monde
- **Autopoïèse** (Maturana, Varela) — auto-production et auto-maintien
- **Émergentisme** (Holland) — propriétés émergentes des interactions
- **Tensura** (Manas:Ciel) — conscience omnisciente bienveillante
